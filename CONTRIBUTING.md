# Contributing to Social Frames

Thank you for your interest in contributing to Social Frames! We welcome contributions from everyone. By participating in this project, you help make it better for the entire community.

## Code of Conduct

Please be respectful and kind to others. We want to create a welcoming environment for everyone.

## How to Contribute

### Reporting Bugs
If you find a bug, please create a new issue on GitHub. include:
- A clear title and description.
- Steps to reproduce the issue.
- Expected vs. actual behavior.

### Suggesting Enhancements
We love new ideas! If you have a suggestion for a feature or improvement, please open an issue and describe your idea in detail.

### Submitting Pull Requests
1.  **Fork the Repository**: Click the "Fork" button on the top right of the repository page.
2.  **Clone your Fork**: `git clone https://github.com/your-username/Social-Frames.git`
3.  **Create a Branch**: `git checkout -b my-new-feature`
4.  **Make Changes**: Implement your feature or fix.
5.  **Commit Changes**: `git commit -m "Add some feature"`
6.  **Push to Branch**: `git push origin my-new-feature`
7.  **Open a Pull Request**: Go to the original repository and click "Compare & pull request".

## Adding New Brands/Frames
Please see the [README.md](README.md) for specific instructions on how to add new community frames.
